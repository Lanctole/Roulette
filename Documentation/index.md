# Добро пожаловать в документацию проекта [Media-Random](https://media-random.ru/)

Проект **Media-Random** — это веб-приложение для случайного выбора контента (игр, аниме и т.д.), которое использует современные технологии, такие как Blazor, ASP.NET Core и Docker.

## Быстрый старт

1. **Регистрация и авторизация**: В проекте реализована система регистрации и авторизации пользователей с подтверждением по email.
2. **Работа с API**: Приложение интегрируется с Shikimori для аниме.
3. **Административная панель**: Доступна только для администраторов, через которую можно загружать и обновлять данные по большому количеству контента.
4. **Документация**: Описание доступных методов можно найти в разделе [API](api/Roulette.html).

## Основные технологии

- **Blazor** для разработки клиентской части.
- **ASP.NET Identity** для управления пользователями.
- **Entity Framework Core** для работы с базой данных.
- **Docker** для контейнеризации и развертывания приложения.
- **NGINX** в качестве обратного прокси-сервера.
