# Планы на будущее

Проект Media-Roulette продолжает активно развиваться, и в ближайших планах следующие улучшения и новые функции:

## 1. Раздел с фильмами и сериалами
Добавление полноценного раздела для управления фильмами и сериалами. Этот функционал будет включать:
- Возможность поиска и фильтрации по жанрам, годам выпуска и рейтингам.
- Сбор данных с внешних API для получения информации о фильмах и сериалах.

## 2. Быстрые переходы на торрент-трекеры
Планируется добавить интеграцию с популярными торрент-трекерами для быстрого перехода к загрузке фильмов, сериалов и игр. Это позволит пользователям:
- Легко и быстро находить ссылки на торрент-файлы.

## 3. Переход на GraphQL при работе с Shikimori API
Планируется переход на использование GraphQL для взаимодействия с Shikimori API.
